## Git Commands

```
git commit -m  "<message>
```
```
git push -u origin master
```
  
```
git push -u origin <branch name>
```
  
### To checkout and create a new Branch

```
git checkout -b <branch name>
```
  
### To see all branches

```
git branch -a
```
